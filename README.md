# Intrucciones y requisitos
Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:
Python 3.9 o superior
pip (administrador de paquetes de Python)
Conexión a internet (para instalar dependencias)
Editor de código recomendado: Visual Studio Code
Puedes verificar tu versión de Python ejecutando en terminal:
python --version

Paso 1
Abre Visual Studio Code.
Selecciona File → Open Folder.
Abre la carpeta raíz del proyecto.
Ejemplo de ruta en Windows:
C:\Users\Sebastian\Downloads\Proyecto_final

Paso 2
Una vez dentro de la ruta del proyecto ejecuta lo siguiente:
pip install -r requirements.txt
esto italara todas las dependencias y librerias necesarias para el proyecto
(todo esto en caso de ya tener python instalado en tu editor de codigo)


Paso 3
Una vez se instalaron los requerimiento
nos aseguramos que estamos en la ruta correcta
y ejecutamos:
python -m streamlit run app.py


Paso 4
El codigo se ejecutara y nos redireccionara autmaticamente al al dashboara 
dentro de nuestro: http://localhost:8501
